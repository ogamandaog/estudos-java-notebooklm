# estudos-java-notebooklm
Projeto de estudo em Java utilizando IA (NotebookLM) para organização de conteúdos, criação de trilha de aprendizado e revisão estruturada.

Este projeto tem como objetivo explorar o uso da Inteligência Artificial, por meio do NotebookLM, como ferramenta de apoio ao estudo de Programação em Java.



A partir da curadoria de aproximadamente 40 materiais didáticos, foi realizada a organização dos conteúdos em uma sequência lógica e pedagógica, considerando níveis de complexidade, pré-requisitos e estratégias de aprendizagem progressiva.



Durante o desenvolvimento, foram elaborados e testados diversos prompts com foco em:

- estruturação de trilhas de estudo;

- identificação de lacunas e redundâncias nos conteúdos;

- criação de planos de estudo progressivos;

- desenvolvimento de estratégias de revisão e retenção;

- aprofundamento do conhecimento em diferentes camadas (teórica, prática e analítica).



O projeto também documenta os resultados obtidos com os prompts, bem como ajustes realizados ao longo do processo, evidenciando o raciocínio aplicado na extração de respostas mais eficazes da IA.



Como entrega final, foi desenvolvido um mini guia de estudos em Java, contendo resumos estruturados, glossário de conceitos e sugestões de prompts reutilizáveis para revisões futuras.



Este trabalho demonstra o uso estratégico da IA como ferramenta de aprendizagem ativa, organização do conhecimento e desenvolvimento de autonomia nos estudos.
